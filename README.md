# Customer Shopping Trends Data Analysis 🛍️

A corporate-grade end-to-end data analytics workflow designed to translate raw consumer data into strategic business intelligence.

## 📌 Project Overview
The goal of this project is to simulate a professional data environment by handling the entire lifecycle of a data project:

* **✅ Data Preparation & EDA (Python):** Cleaning and transforming raw datasets to ensure data integrity.
* **✅ Data Analysis (SQL):** Simulating business transactions and querying complex datasets to extract insights on loyalty and purchase drivers.
* **✅ Visualization (Power BI):** Developing an interactive dashboard to highlight patterns and trends for stakeholders.
* **✅ Reporting & Presentation:** Summarizing findings into actionable business recommendations using professional documentation and AI-assisted presentation tools.

## 🛠️ Tech Stack
- **Languages:** Python (Pandas, SQLAlchemy), SQL (MySQL/PostgreSQL)
- **Visualization:** Power BI
- **Documentation:** Markdown, MS Word
- **Presentation:** Gamma AI
